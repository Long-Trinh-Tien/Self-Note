Giả sử local có folder Christmas
	Xoá file (list name truyền vào):rclone delete archive:/MUSIC-YT-TMY -P --files-from <(rclone lsf "Christmas")
	Kiểm tra file có trong list không: rclone copy "Nhạc/" 'archive:/MUSIC-YT-TMY/Nhac' --dry-run
	Copy từ local lên: rclone copy "Old song usuk" 'archive:/MUSIC-YT-TMY/Old_song_usuk' --include "*.mp3" -P
		list command: 
			rclone copy "Old song usuk" 'archive:/MUSIC-YT-TMY/Old_song_usuk' --include "*.mp3" -P
			rclone copy "Nhạc" 'archive:/MUSIC-YT-TMY/Nhac' --include "*.mp3" -P
			rclone copy "Christmas" 'archive:/MUSIC-YT-TMY/Christmas' --include "*.mp3" -P
			rclone copy "Relax không lời" 'archive:/MUSIC-YT-TMY/Relax không lời' --include "*.mp3" -P
			rclone copy "Remix" 'archive:/MUSIC-YT-TMY/Remix' --include "*.mp3" -P
