- Cài vào usb
- Mount usb vào /mnt
- Boot vào ubuntu: sudo grub-install --target=i386-pc --boot-directory=/mnt/ /dev/sdX
- Tạo /mnt/grub/grub.cfg
	set root=(hd1,msdos5)
	set prefix=(hd1,msdos5)/boot/grub
	insmod normal
	normal
-> để grub của ubuntu có sẵn xử lý tiếp
